== Description ==  
??? **Event QR Code Registration & Check-In** is a powerful WordPress plugin that allows you to **register attendees, generate QR codes, and enable contactless check-ins** for events.  
Ideal for **conferences, workshops, ticketed events, and corporate meetings**, this plugin provides a **seamless way to manage guests**.  

? **Key Features:**  
? **Generate unique QR codes** for every attendee  
? **Contactless event check-in system** via QR scanning  
? **Track attendance** & prevent duplicate check-ins  
? **Admin dashboard** to manage attendee lists  
? **Mobile-friendly & responsive design**  
? **Shortcode support** – Easy to embed on any page  

?? **Perfect for:** Business Events, Conferences, Workshops, Webinars, Trade Shows, and More! ??  

== Installation ==  
1. **Download & Install:** Upload the plugin to `/wp-content/plugins/event-qr-registration`.  
2. **Activate:** Go to **Plugins > Activate**.  
3. **Use the Shortcodes:**  
   - Add `[event_qr_form]` to display the registration form.  
   - Add `[event_qr_verification]` to allow QR code check-ins.  

== Frequently Asked Questions ==  

= How does the QR code system work? =  
When a user registers for an event, they receive a **unique QR code**. This QR code is scanned at the event for seamless check-in.  

= Can I scan QR codes with my mobile phone? =  
Yes! You can use **any QR code scanner app** or your phone’s built-in camera to scan attendee QR codes.  

= Can I track check-ins in real-time? =  
Yes, the **admin dashboard** lets you **track attendees, view check-in history, and manage registrations**.  

= Does this work with WooCommerce or other ticketing plugins? =  
Currently, this plugin operates independently. Future versions may include WooCommerce integration.  

= Is the registration form customizable? =  
Yes, you can customize form fields or use CSS to style the form to match your website.
